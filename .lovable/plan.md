### המלצה

**אם רוצה לתעדף:** עשה רק את 3 הבסיסיים + קיצור מקלדת + נקה שיחה = **~0.6 קרדיט**. זה נותן 90% מהערך. שמירת מיקום ואנימציית מינימוז אפשר להוסיף אחר כך.

לשם השוואה — תיקון הרווחים שעשינו קודם היה ~1.0 קרדיט, וזה היה הרבה יותר מורכב (2 קבצים, בדיקות API חיות).
